# Project Spatia: Glossary of Terms

- **Atom**: The smallest unit of content-addressed logic (function/struct).
- **Anchor**: A fixed coordinate {x, y} in the tmux workspace.
- **Geometric Lisp (Slang B)**: The S-expression language defining spatial metadata.
- **Envelope**: A safety boundary or policy wrapper around Atoms.
- **Portal**: A live, read-only link to external truth (Nix store/hardware specs).
- **Thread**: A visual Unicode line connecting Intent to Implementation.
- **The Witness**: A Docker-based verification agent for AI "Claims."
- **The Sentinel**: The SQLite sidecar daemon managing the graph.
